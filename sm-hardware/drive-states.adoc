= 了解 SANtricity System Manager 中的磁碟機狀態
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
SANtricity System Manager 會報告磁碟機的各種狀態。



== 無障礙狀態

[cols="25h,~"]
|===
| 狀態 | 定義 


 a| 
已略過
 a| 
磁碟機實際存在，但控制器無法透過任一連接埠與其通訊。



 a| 
不相容
 a| 
存在以下條件之一：

* 此磁碟機未經認證，無法在儲存陣列中使用。
* 磁碟機的磁區大小不同。
* 此磁碟機包含來自較舊或較新韌體版本的不可用組態資料。




 a| 
已移除
 a| 
該磁碟機已從儲存陣列中不當移除。



 a| 
目前
 a| 
控制器可以透過兩個連接埠與磁碟機通訊。



 a| 
無回應
 a| 
磁碟機沒有回應命令。

|===


== 角色狀態

[cols="25h,~"]
|===
| 狀態 | 定義 


 a| 
已分配
 a| 
此磁碟機是資源池或磁碟區群組的成員。



 a| 
使用中的熱備援
 a| 
該磁碟機目前用作故障磁碟機的替代品。熱備援磁碟機僅用於磁碟區群組，不用於儲存池。



 a| 
待命熱備援
 a| 
該磁碟機已準備就緒，可用作故障磁碟機的替代品。熱備援磁碟機僅用於磁碟區群組，不用於儲存池。



 a| 
未分配
 a| 
此磁碟機不屬於任何儲存池或磁碟區群組。

|===


== 可用性狀態

[cols="25h,~"]
|===
| 狀態 | 定義 


 a| 
失敗
 a| 
磁碟機無法運作。磁碟機上的資料無法使用。



 a| 
即將故障
 a| 
系統偵測到該磁碟機可能很快就會發生故障。磁碟機上的資料仍然可用。



 a| 
離線
 a| 
通常情況下，該磁碟機無法用於儲存資料，因為它屬於正在匯出的磁碟區群組，或正在進行韌體升級。



 a| 
最佳
 a| 
磁碟機運作正常。

|===
