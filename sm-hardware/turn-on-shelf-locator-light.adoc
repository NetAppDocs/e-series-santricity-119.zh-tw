= 在 SANtricity System Manager 中開啟機櫃定位燈
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
若要尋找 Hardware 頁面上顯示的機櫃的實際位置、您可以開啟機櫃的定位燈。

.步驟
. 選擇 *Hardware* 。
. 選取 Controller Shelf 或 Drive Shelf 的下拉式清單，然後選取 *Turn on locator light* 。
+
貨架的定位燈亮起。

. 當您實際找到磁碟櫃後，請返回對話方塊並選擇 *關閉*。
