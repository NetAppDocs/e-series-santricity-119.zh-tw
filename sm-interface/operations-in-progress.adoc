= 了解 SANtricity System Manager 操作進行中頁面
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
「進行中的操作」頁面可讓您檢視目前正在執行或處於暫停、停止或失敗狀態的長時間執行操作清單。您可以對某些類型的操作採取動作。

長時間執行的作業可能會影響系統效能。對於某些作業，您可以選擇停止作業或變更其優先順序，以便更重要的作業優先完成。
