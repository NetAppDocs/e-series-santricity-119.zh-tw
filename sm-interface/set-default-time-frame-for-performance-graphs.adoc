= 在 SANtricity System Manager 中設定效能圖表的預設時間範圍
:allow-uri-read: 
:experimental: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以變更效能圖表顯示的預設時間範圍。

.關於此任務
首頁和效能頁面上顯示的效能圖表最初顯示的是 1 小時的時間範圍。偏好設定儲存在瀏覽器的本機儲存中，因此所有使用者都可以擁有自己的設定。

.步驟
. 選擇選單：Preferences [ Set preferences ]。
. 在下拉清單中、選取 *5 分鐘*、*1 小時*、*8 小時*、*1 天* 或 *7 天*、然後確認您要執行此作業。
