= 了解 SANtricity 軟體中的非同步鏡像功能
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
非同步鏡像功能提供了一種基於控制器級韌體的機制，用於在本機儲存陣列和遠端儲存陣列之間進行資料複製。



== 什麼是非同步鏡像？

_非同步鏡像_ 會在特定時間點擷取主磁碟區的狀態，並僅複製自上次映像擷取以來發生變更的資料。主站台可以立即更新，而次要站台則可根據頻寬情況進行更新。資訊會被快取，並在網路資源可用時稍後傳送。

非同步鏡像以磁碟區為單位建立，但在群組層級進行管理，可讓您將獨立的遠端鏡像磁碟區與給定儲存陣列上的任何主磁碟區建立關聯。這種鏡像類型非常適合滿足不間斷作業的需求，而且一般來說，對於週期性程序的網路效率更高。

== 如何配置非同步鏡像？

您必須使用 Unified Manager 介面來執行陣列之間的初始鏡像配置。配置完成後，您可以在 System Manager 中管理鏡像對和一致性群組。
