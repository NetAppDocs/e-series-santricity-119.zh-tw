= 在 SANtricity System Manager 中檢視鏡像一致性群組之間未同步的資料量
:allow-uri-read: 
:experimental: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以查看本機儲存陣列和遠端儲存陣列上鏡像一致性群組之間未同步的資料量。當鏡像一致性群組處於未同步狀態時，不會進行任何鏡像活動。

.關於此任務
當選取的鏡像一致性群組包含鏡像配對且目前未進行同步時、您可以執行此工作。

.步驟
. 選擇功能表：Storage[Asynchronous Mirroring]。
. 選擇 *Mirror Consistency Groups* 標籤。
+
鏡像一致性群組表隨即出現，並顯示與儲存陣列關聯的所有鏡像一致性群組。

. 點選選單：更多 [檢視未同步資料量]。
+
如果存在未同步數據，表中的值會反映這一點。「資料量」列列出了未同步資料的數量，單位為 MiB。
