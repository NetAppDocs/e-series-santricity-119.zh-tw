= 在 SANtricity System Manager 中移除 SNMPv3 陷阱的使用者
:allow-uri-read: 
:experimental: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以移除 SNMPv3 陷阱的使用者。

.步驟
. 選擇選單：Settings [ Alerts ]。
. 選擇 *SNMP* 選項卡。
+
陷阱目的地和使用者名稱會顯示在警示頁面上。

. 選擇 *Configure Users* 。
+
Configure SNMPv3 Users 對話方塊隨即開啟。

. 選擇要刪除的使用者名稱，然後按一下 *Delete* 。
. 確認操作，然後按一下 *Delete* 。


.結果
使用者名稱及其關聯的設陷目的地將從「警示」頁面中移除。
