= 在 SANtricity System Manager 中開啟儲存陣列定位器指示燈
:allow-uri-read: 
:experimental: 
:icons: font
:imagesdir: ../media/


[role="lead"]
若要找到儲存陣列在機櫃中的實體位置、您可以開啟其定位器 (LED) 燈。

.步驟
. 選擇功能表：Settings[System]。
. 在 *General* 下，按一下 *Turn on Storage Array Locator Lights* 。
+
「開啟儲存陣列定位燈」對話方塊隨即開啟，對應的儲存陣列定位燈也會亮起。

. 當您實際找到儲存陣列後，返回對話方塊並選擇 *Turn Off* 。


.結果
定位器指示燈熄滅，對話方塊關閉。
