= 在 SANtricity System Manager 中啟動和暫停快照排程
:allow-uri-read: 
:experimental: 
:icons: font
:imagesdir: ../media/


[role="lead"]
當您需要節省儲存空間時，可以暫時中止排程收集快照映像。此方法比刪除快照排程後再重新建立更有效率。

.關於此任務
快照排程的狀態將保持暫停狀態，直到您使用 *Activate* 選項恢復排程的快照活動。

.步驟
. 選擇選單：Storage [ Snapshots ]。
. 如果尚未顯示，請點選 *Schedules* 標籤。
+
排程已列在頁面上。

. 選擇要暫停的作用中快照排程，然後按一下 *Activate / Suspend*。
+
State 欄狀態變更為 *Suspended*，快照排程會停止收集所有快照映像。

. 若要恢復收集快照影像，請選擇要復原的已暫停快照排程，然後按一下 *啟動 / 暫停*。
+
State 欄的狀態變更為 *Active*。
