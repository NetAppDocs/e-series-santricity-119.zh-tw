= 在 SANtricity System Manager 中變更主機叢集的設定
:allow-uri-read: 
:experimental: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以變更主機叢集名稱，或在主機叢集中新增或移除主機。

.步驟
. 選擇選單：Storage[Hosts]。
. 選擇要編輯的主機叢集，然後按一下 *View/Edit Settings* 。
+
此時會出現一個對話方塊，顯示目前的主機叢集設定。

. 根據需要變更主機叢集的設定。
+
.欄位詳細資料
[%collapsible]
====
[cols="25h,~"]
|===
| 設定 | 說明 


 a| 
Name
 a| 
您可以指定主機叢集的使用者自訂名稱。必須指定叢集名稱。



 a| 
相關主機
 a| 
若要新增主機，請按一下 *Associated Hosts* 方塊，然後從下拉式清單中選取主機名稱。您無法手動輸入主機名稱。

若要刪除主機，請按一下主機名稱旁邊的 *X*。

|===
====
. 按一下 *Save* 。
