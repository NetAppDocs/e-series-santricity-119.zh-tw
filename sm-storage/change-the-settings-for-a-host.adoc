= 在 SANtricity System Manager 中變更主機的設定
:allow-uri-read: 
:experimental: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以變更主機的名稱、主機作業系統類型以及相關的主機叢集。

.步驟
. 選擇選單：Storage[Hosts]。
. 選取要編輯的主機，然後按一下 *View/Edit Settings* 。
+
此時會出現一個對話方塊，顯示目前的主機設定。

. 如果尚未選取，請按一下 *Properties* 標籤。
. 根據需要變更設定。
+
.欄位詳細資料
[%collapsible]
====
[cols="25h,~"]
|===
| 設定 | 說明 


 a| 
Name
 a| 
您可以變更使用者提供的主機名稱。必須指定主機名稱。



 a| 
相關主機叢集
 a| 
您可以選擇以下選項之一：

** *None* -- 主機仍為獨立主機。如果主機已關聯到主機叢集，系統會將該主機從叢集中移除。
** *<Host Cluster>* -- 系統將主機與選定的叢集建立關聯。




 a| 
主機作業系統類型
 a| 
您可以變更在您定義的主機上執行的作業系統類型。

|===
====
. 按一下 *Save* 。
