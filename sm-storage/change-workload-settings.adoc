= 在 SANtricity System Manager 中變更工作負載設定
:allow-uri-read: 
:experimental: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以變更工作負載的名稱並查看其關聯的應用程式類型。當目前名稱不再有意義或適用時，請變更工作負載的名稱。

.步驟
. 選擇選單：Storage [ Volumes ]。
. 選擇 *Applications & Workloads* 標籤。
+
顯示 Applications & Workloads 視圖。

. 選取您要變更的工作負載，然後選取 *View/Edit Settings* 。
+
應用程式與工作負載設定對話方塊隨即出現。

. *選用：* 變更使用者提供的工作負載名稱。
. 按一下 *Save* 。
