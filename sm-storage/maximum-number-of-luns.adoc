= 了解 SANtricity 軟體中 LUN 的最大數量
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
儲存陣列對每個主機可使用的邏輯單元號碼 (LUN) 數量有上限。

最大數量取決於主機的作業系統。儲存陣列會追蹤已使用的 LUN 數量。如果您嘗試將磁碟區指派給超過 LUN 數量上限的主機，該主機將無法存取該磁碟區。
