= 在 SANtricity System Manager 中取消指派磁碟區
:allow-uri-read: 
:experimental: 
:icons: font
:imagesdir: ../media/


[role="lead"]
如果不再需要從主機或主機叢集對該磁碟區進行 I/O 存取，請從主機或主機叢集取消指派磁碟區。

.關於此任務
取消分配磁碟區時，請牢記以下準則：

* 如果您要從主機叢集中移除最後一個指派的磁碟區，且該主機叢集中還有具有特定指派磁碟區的主機，請確保在移除主機叢集的最後一個指派之前，先移除或移動這些指派。
* 如果主機叢集、主機或主機連接埠已指派給已註冊到作業系統的磁碟區，則必須先清除此註冊才能刪除這些節點。


.步驟
. 選擇選單：Storage[Hosts]。
. 選擇要編輯的主機或主機叢集，然後按一下 *取消指派磁碟區*。
+
此時會出現對話方塊，顯示目前已指派的所有磁碟區。

. 選取要取消指派的每個磁碟區旁的核取方塊，或選取表格標題中的核取方塊以選取所有磁碟區。
. 點選 *Unassign* 。


.結果
* 未指派的磁碟區可供新的指派。
* 在主機上完成變更設定之前，主機作業系統仍然可以識別該磁碟區。
