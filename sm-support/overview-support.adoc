= 在 SANtricity System Manager 中取得支援
:allow-uri-read: 
:experimental: 
:icons: font
:imagesdir: ../media/


[role="lead"]
支援頁面可讓您存取技術支援資源。



== 有哪些 Support 任務可供選擇？

在「支援」中、您可以檢視技術支援聯絡人、執行診斷、設定 AutoSupport、檢視事件記錄、以及執行軟體升級。

深入瞭解：

* link:autosupport-feature-overview.html["AutoSupport 功能概述"]
* link:overview-event-log.html["事件記錄概觀"]
* link:overview-upgrade-center.html["升級中心概覽"]




== 我該如何聯絡技術支援？

在主頁上，點選功能表：Support[Support Center > Support Resources 標籤]。技術支援聯絡資訊列在介面右上角。
