= 在 SANtricity System Manager 中擷取追蹤緩衝區
:allow-uri-read: 
:experimental: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以從控制器中擷取追蹤緩衝區，並將檔案傳送給技術支援進行分析。

.關於此任務
韌體使用追蹤緩衝區記錄處理過程，特別是異常情況，這有助於除錯。您可以擷取追蹤緩衝區，而不會中斷儲存陣列的運作，並且對效能的影響也極小。

[NOTE]
====
僅在技術支援人員指示時才執行此操作。

====
.步驟
. 選擇功能表：Support [ Support Center > Diagnostics ] 標籤。
. 選擇 *Retrieve Trace Buffers* 。
. 選取要擷取追蹤緩衝區之每個控制器旁的核取方塊。
+
您可以選擇一個或兩個控制器。如果核取方塊右側的控制器狀態訊息為 Failed 或 Disabled、則會停用該核取方塊。

. 點選 *Yes* 。
+
檔案會以檔案名稱  `trace-buffers.7z` 儲存在瀏覽器的「下載」資料夾中。

. 請按照技術支援人員提供的說明將檔案傳送給他們。
