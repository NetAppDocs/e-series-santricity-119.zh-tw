= 在 SANtricity Unified Manager 中清除分階段作業系統軟體
:allow-uri-read: 
:experimental: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以移除已暫存的 OS 軟體，以確保待定版本不會在稍後被意外啟動。移除已暫存的 OS 軟體不會影響儲存陣列上目前執行的版本。

.步驟
. 從主視圖中選擇 *Manage*，然後選擇 menu:Upgrade Center[Clear Staged OS Software]。
+
「清除暫存作業系統軟體」對話方塊隨即開啟，並列出所有已探索到且具有擱置中軟體或 NVSRAM 的儲存系統。

. 如有需要，可對清單中的儲存系統進行篩選或排序，以便查看所有已部署軟體的系統。
. 選取要清除待處理軟體的儲存系統核取方塊。
. 點選 *Clear* 。
+
操作狀態顯示在對話方塊中。
