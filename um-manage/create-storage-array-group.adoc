= 在 SANtricity Unified Manager 中設定儲存陣列群組
:allow-uri-read: 
:experimental: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以建立儲存群組，然後將儲存陣列新增至群組。

配置群組是一個兩步驟程序。



== 步驟 1：建立群組

首先，您需要建立一個群組。儲存群組定義了哪些磁碟機提供構成該磁碟區的儲存空間。

.步驟
. 從「管理」頁面，選取功能表：管理群組 [ 建立儲存陣列群組 ] 。
. 在 *Name* 欄位中、輸入新群組的名稱。
. 選擇要新增至新群組的儲存陣列。
. 按一下 *Create* 。




== 步驟 2：將儲存陣列新增至群組

您可以將一個或多個儲存陣列新增至使用者建立的群組。

.步驟
. 從主視圖中選擇 *Manage*，然後選擇要新增儲存陣列的群組。
. 選擇選單：Manage Groups[Add storage arrays to group]。
. 選擇要新增至群組的儲存陣列。
. 按一下 *Add* 。
